🚀 Cẩm Nang GitHub Pages

Hướng dẫn từ A-Z cách đưa website tĩnh của bạn lên internet hoàn toàn miễn phí.

Bước 1: Tạo kho lưu trữ (Repository) mới trên GitHub

Đăng nhập vào GitHub. Nhấn nút New để tạo một Repository mới.

  • Repository name: Đặt tên chính xác theo định dạng tencuaban.github.io (trong đó tencuaban là username GitHub của bạn).
  • Chọn Public (Công khai).
  • Nhấn Create repository.

Bước 2: Chuẩn bị mã nguồn Website

Tạo các file mã nguồn của bạn (như trang web này với index.html, style.css, script.js) trên máy tính.

Lưu ý: Trang chính của bạn bắt buộc phải có tên là index.html.

Bước 3: Tải mã nguồn lên GitHub

Bạn có thể sử dụng Git trên Terminal/Command Line hoặc đơn giản nhất là kéo thả trực tiếp trên giao diện web của GitHub:

  1. Vào Repository vừa tạo.
  2. Nhấp vào liên kết uploading an existing file.
  3. Kéo thả các file web của bạn vào và nhấn Commit changes.

Bước 4: Tận hưởng thành quả

Đợi khoảng 1-5 phút để GitHub xử lý. Sau đó, hãy mở trình duyệt và truy cập vào địa chỉ: https://tencuaban.github.io.

🎉 Chúc mừng! Trang web của bạn đã online.
